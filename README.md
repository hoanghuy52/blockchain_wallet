# blockchain_wallet
Simple Blockchain Wallet

Hoàn thành phần back-end căn bản, chưa hoàn thành GUI
Back-end:
	+ Proof of Work
	+ Key-Pair (ECC)
	+ Rebroadcast Transaction
	+ Peer to Peer - Connect
	+ Create Account
	+ Mine
